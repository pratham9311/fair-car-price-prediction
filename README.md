# 🚗 Fair Car – AI-Based Used Car Price Prediction

## 📌 Overview

Fair Car is a machine learning-based web application designed to predict the fair market price of used cars. The system uses historical data and multiple vehicle attributes to generate accurate and reliable price estimations, reducing guesswork and improving transparency in the used car market.

---

## 🎯 Objective

* Develop an AI-based system to estimate used car prices
* Improve accuracy and consistency compared to manual valuation
* Provide fair and data-driven price suggestions for buyers and sellers

---

## ⚙️ Features

* Predicts car price based on input parameters
* Uses Machine Learning (Random Forest Regression)
* User-friendly interface (Web-based)
* Data preprocessing and feature engineering
* Model evaluation using MAE, RMSE, and R²

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Streamlit / Flask (for deployment)
* Google Colab

---

## 📊 Input Parameters

The model predicts price based on:

* Car Name / Model
* Location
* Year of Manufacture
* Kilometers Driven
* Fuel Type
* Transmission
* Owner Type
* Mileage
* Engine Capacity
* Power
* Number of Seats

---

## 📈 Machine Learning Model

* Algorithm: Random Forest Regression
* Approach: Supervised Learning
* Evaluation Metrics:

  * Mean Absolute Error (MAE)
  * Root Mean Squared Error (RMSE)
  * R-squared (R²)

## 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/fair-car-price-prediction.git
   ```

2. Navigate to the project folder:

   ```bash
   cd fair-car-price-prediction
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:

   ```bash
   streamlit run app.py
   ```

   or

   ```bash
   python app.py
   ```
